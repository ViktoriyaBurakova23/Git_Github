Основные команды GIT

git init - создание нового репозитория в рабочем каталоге;

git clone git@github.com:логин/название репозитория.git - создание локальной рабочей копии репозитория;

git config –-global user.name “name”
           –-global user.email “email” - устанавливает имя пользователя и адрес электронной почты;

git add file.txt - добавление файла в область подготовки (Staging Area/Index содержит информацию о том, что должно войти в последующий коммит);

git add . - добавление всех файлов в область подготовки;

git commit -m "Описание коммита" - фиксирует состояние файла в локальном репозитории с добавлением комментария;

git status - показывает текущее состояние файлов;

git rm <file_name> - удаление файла;

git mv <file_name1> <fle_name2> - переименование файла;

git branch - выводит список всех веток в локальном репозитории;

git checkout -b <branch_name> - создание новой ветки и переключение на нее;

git checkout <branch_name2> - переход в другую ветку;

git branch -d <branch_name> - удаление ветки;

git merge <branch_name> - используется для слияния ветки с изменениями с главной (принимающей);

git push - отправляет зафиксированные изменения в удаленный репозиторий;

git push -u origin <branch_name> - отправляет зафиксированные изменения определенной ветки в удаленный репозиторий;

git pull - забирает изменения с удаленного репозитория в локальный.
